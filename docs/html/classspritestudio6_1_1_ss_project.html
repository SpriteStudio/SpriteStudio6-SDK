Detailed Description

XMLドキュメントとなっているsspjファイルのデータ保持を提供するクラスです。 以下はエディタ情報のため読み飛ばします。 animeSettings デフォルトのアニメーション設定 texPackSettings デフォルトのテクスチャパッキング設定
